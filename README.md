# ProcureIntel

Procurement intelligence platform for discovering, tracking, and analyzing tenders.

## Features
- Tender discovery and watchlist
- Keyword alerts
- Notifications
- Bid comparison
- Supplier matching
- Admin panel

## Setup
1. Copy `.env.example` to `.env`
2. Install dependencies with `pnpm install`
3. Run the app workflows

## Environment
- `PORT`
- `NODE_ENV`
- `LOG_LEVEL`
- `DATABASE_URL`
- `JWT_SECRET`
- `VITE_API_BASE_URL`
